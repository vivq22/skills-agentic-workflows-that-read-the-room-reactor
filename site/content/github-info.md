## Latest GitHub Updates

- **Keep dormant repos scanned for security:** GitHub code scanning can now run periodically on repositories with no pushes or pull requests for 6+ months, helping teams keep long-lived repos covered. *(Source: GitHub Changelog, 2026-06-09 — "Periodic code scanning of inactive repositories")*
- **Add a security gate for third-party agent code:** Code generated by third-party coding agents now gets automatic security and quality validation, which makes agent adoption safer in shared repos. *(Source: GitHub Changelog, 2026-06-09 — "Security validation for third-party coding agents")*
- **Watch the new Copilot desktop surface for agent workflows:** GitHub introduced the GitHub Copilot app as an agent-native desktop experience, signaling more end-to-end agent workflows beyond the browser and IDE. *(Source: GitHub Blog, 2026-06-02 — "GitHub Copilot app: The agent-native desktop experience")*
